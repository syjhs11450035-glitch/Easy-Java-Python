# 📦 變數宣告：sv (Set Variable)

`sv` 是 EJP 最基本的指令，代表「設定變數」。

## 語法結構
```EJP
sv( 變數名稱 , 數值 );
```

## 應用範例
1. **數字儲存**：`sv(%"apple"%, 10);` (現在 apple 代表 10)
2. **文字儲存**：`sv(%"name"%, "Kun Xu Liu");`
3. **混合運算**：`sv(%"total"%, apple % 2);`

> 💡 **小撇步**：在 EJP 中，變數不需要宣告型別（不需要寫 int 或 String），引擎會自動幫你判斷！
