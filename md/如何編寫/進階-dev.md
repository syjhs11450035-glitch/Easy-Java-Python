# 🛠️ 開發者模式 (Developer Mode)

專門為 debug（除錯）與深入探索引擎設計的高級工具。

## 1. 啟動 dev 模式
當你在代碼中使用 dev 指令，EJP 會進入特殊的偵錯狀態。
```EJP
dev("dev名稱", %"dav內變數名稱"%){};

## 使用 ##
dev名稱(1)

```
> **功能說明**：在此模式下，控制台會顯示更多隱藏的轉譯細節，包括 6-bit 編碼的轉換過程以及變數在記憶體中的即時變化。

## 2. 除錯的好幫手
如果你發現程式執行結果不如預期，開啟 dev 模式能幫你精確定位是哪一行邏輯出了問題。
