# 📢 顯示與註解 (Logging & Comments)

在編寫程式時，與電腦對話以及幫自己寫筆記是非常重要的。

## 1. 顯示訊息：log()
當你想讓電腦在控制台（Console）輸出文字或變數內容時，請使用 log 指令。
```EJP
log("文字");
```
> **用法說明**：在括號內放入你想顯示的內容。如果是純文字，請用雙引號包起來；如果是變數名稱，則直接放入即可。記得結尾要加上分號。

## 2. 撰寫註解：##
註解是給人類看的筆記，電腦會完全忽略這些內容。
```EJP
## 我是註解 ##
```
> **用法說明**：在任何一行指令的開頭或結尾加上雙井字號，後方跟隨的文字就不會被執行。這對於標記代碼功能或暫時停用某行指令非常有用。
