
# 🪄 多語言混寫 (Polyglot)

EJP 最引以為傲的功能，就是能在代碼中直接召喚其他語言的神力。

## 1. Python 塊：py{}
當 EJP 的語法不夠用時，你可以直接寫純 Python 代碼。
```ejp
py{
   import random
   print(random.randint(1, 100))
}

```

## 2. Java 與 C 塊 (預覽功能)

* **java{}**：未來將支援直接調用 JVM 執行聯邦級別的運算。
* **c{}**：用於底層硬體控制，實現極致速度。
