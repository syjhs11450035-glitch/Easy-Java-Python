## 1. 系統需求 (System Requirements)

在開始安裝之前，請確認您的電腦符合以下條件：
* **作業系統**: Windows 10/11, macOS, 或 Linux。
* **Python 環境**: 必須安裝 **Python 3.8** 或更高版本。
* **硬體空間**: 至少 50MB 的可用磁碟空間（主要用於存放文檔與 Logo 圖片）。

## 2. 環境配置 (Environment Setup)

### 2.1 安裝 Python 依賴庫
EJP IDE 本地版使用 `Tkinter` 建立圖形介面，並使用 `Pillow` 處理圖像。請打開終端機（cmd 或 terminal）執行以下指令：

```bash
pip install pillow
```

### 2.2 獲取 EJP

目前階段，您可以透過以下方式獲取：

1. **源碼下載**: 直接從官方儲存庫下載 `run.py` 與 `logo/` 資料夾。
2. **預編譯包 (開發中)**: 未來我們將提供獨立的 `.exe` 或 `.dmg` 安裝檔，讓您無需安裝 Python 即可執行。

## 3. 啟動方式 (How to Launch)

1. **檢查檔案**: 確認 `run.py`存在。
2. **運行**:
* Windows: 雙擊 `run.py` 或在 cmd 輸入 `python run.py`。
* macOS/Linux: 在終端機輸入 `python3 run.py`。

## 4. 常見問題 (FAQ)

* **Q: 為什麼畫布 (Canvas) 沒有彈出來？**
* A: 畫布視窗會在您代碼中第一次執行 `canva(add, ...)` 指令時自動實體化。


* **Q: pip 指令無效？**
* A: 請確保在安裝 Python 時有勾選「Add Python to PATH」選項。
